Teknisk SEO: gjør siden din synlig og rask

Teknisk SEO handler om å hjelpe søkemotorer å finne, lese og indeksere innholdet ditt. Sjekk at du bruker HTTPS, har ryddige URL-er og at viktige ressurser (CSS/JS/bilder) ikke blokkeres. En XML-sitemap og en korrekt robots.txt gjør jobben enklere for Google. Optimaliser lastetid for bedre brukeropplevelse og rangering.

  • Bruk Google Search Console til å overvåke indeksering og feil
  • Forbedre hastighet og stabilitet (Core Web Vitals)
  • Unngå duplikatinnhold med rel="canonical" og riktige omdirigeringer

Titler og metabeskrivelser som gir klikk

Side-tittelen (HTML-<title>) og metabeskrivelsen avgjør om brukeren klikker på resultatet ditt. Skriv unike, presise titler (ca. 50–60 tegn) og korte, selgende beskrivelser (ca. 140–155 tegn) som oppsummerer siden. Bruk språk fra målgruppen – ikke nøkkelord-fylling.

Eksempel på god tittel: “Medisinsk yoga i Fredrikstad – timer for stress og søvn”. Eksempel på god metabeskrivelse: “Rolige gruppetimer med fokus på pust og balanse. Påmelding og priser – se ledige plasser.”

Bilder, alt-tekst og filnavn

Bilder kan gi trafikk fra bildesøk og forsterker forståelsen av innholdet. Bruk beskrivende filnavn (f.eks. medisinsk-yoga-instruktor.jpg), legg inn alt-tekst som forklarer motiv og funksjon, og plasser bilder nær relevant tekst. Komprimer bilder og bruk moderne formater der det passer.

Internlenking og informasjonsarkitektur

En tydelig struktur hjelper både brukere og søkemotorer. Sett opp logiske kategorier og bruk interne lenker med beskrivende ankertekst (f.eks. “se priser for medisinsk yoga” fremfor “klikk her”). Viktige sider bør være tilgjengelige på få klikk fra forsiden.

Strukturert data (schema.org) og rike resultater

Med JSON-LD kan du beskrive hva siden handler om (artikkel, arrangement, produkt) slik at Google kan vise rike resultater som stjerner, priser og FAQ. Merk kun opp informasjon som faktisk finnes på siden, og test i Rich Results Test før publisering.
